## Sobre mim

<div style="background:rgba(141,110,99,0.08); padding:18px; border-radius:14px; border:1px solid rgba(215,184,153,0.35); line-height:1.6;">

Desenvolvedor em formação na área de sistemas embarcados e automação industrial, com foco em firmware em C para ESP32 utilizando ESP-IDF e FreeRTOS. Experiência na implementação de lógica de controle baseada em máquina de estados, manipulação de GPIO e sistemas em tempo real.

Conhecimento também em lógica de programação para CLPs utilizando Ladder Logic, integrando conceitos de automação industrial com sistemas embarcados. Atualmente em aprendizado de Python com foco em automação e aplicações técnicas.
